# Portfolio Web Project

## Description

This project is a responsive personal portfolio built with HTML, CSS, and JavaScript. It demonstrates basic front-end development skills, including layout, responsiveness, and interactivity.

# Technologies

- HTML5

- CSS3 (Flexbox, Grid, Media Queries)

- JavaScript (ES6)

# Features

- Responsive navigation menu

- Projects section with adaptive grid layout

- Pets gallery with responsive image grid

- Interactive button using JavaScript

- Welcome message on page load

# Responsiveness

- Mobile: one-column layout, two-column gallery

- Tablet: two-column projects, three-column gallery

- Desktop: multi-column layout
